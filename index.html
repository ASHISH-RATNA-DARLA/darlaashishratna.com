Recent Projects

Aetos Angelos

A keen perception alert system.

Aetos Angelos" is an innovative emergency alert system that uses deep learning technology to enhance the efficiency and accessibility of emergency responses. The system utilizes existing CCTV cameras to detect specific hand signals associated with distress, such as gestures for police, ambulance, or emergency support. By recognizing these signals, "Aetos Angelos" can immediately alert the appropriate emergency services with precise location details, ensuring a swift response.

Wireless Sound Control

This project develops a gesture-based system for wireless control of sound parameters, enhancing user interaction with sound systems through hands-free commands. It uses Python, OpenCV, and MediaPipe for real-time hand gesture recognition to adjust volume or select tracks. The system improves accessibility and hygiene, benefiting users with mobility issues and in public settings.
